**RUN LOCAL SERVER**

  **INSTALLATION**
 ```
  install composer globally
  run over the root composer install
 ```

 **RUN A LOCAL CONTAINER**
 
**_mac users_**
```
 sh run-container.sh
 OPEN http://localhost:4000/
```

**_windows users_**
```
 php -S 0.0.0.0:4000 -t public
 OPEN http://localhost:4000/
```

**Stop Local Container**
```
In terminal 
CTRL + c
```
